# STIT Spring Boot Notes

A beginner-friendly collection of Spring Boot learning notes with simple explanations and examples.

## 📚 About This Repository

This repository contains notes about important Spring Boot concepts such as annotations, REST API development, JSON handling, exception handling, logging, security, transactions, IoC, and Dependency Injection.

The purpose is to make Spring Boot concepts easier to understand in a clear and simple way.

## 📌 Topics Covered

- Spring Boot Basics
- How Spring Boot Works
- Spring Boot Annotations
- REST Controllers
- API Endpoints
- JSON Handling
- Exception Handling
- Logging
- Security (JWT & RBAC)
- Transactions
- IoC Container
- Dependency Injection
