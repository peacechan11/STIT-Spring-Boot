# 🔐 Spring Security Config Notes (with Diagram)

## 📌 Overview

SecurityConfig ဆိုတာက Spring Boot project မှာ  
👉 Authentication (login)  
👉 Authorization (access control)  
ကို control လုပ်တဲ့ configuration class ဖြစ်ပါတယ်။

---

## 🧠 Core Flow Diagram

```
Request → Security Filter Chain → Check URL + Role → Allow / Deny
```

---

## ⚙️ Main Configuration

```java
@Bean
public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
    http
        .csrf(csrf -> csrf.disable())
        .authorizeHttpRequests(auth -> auth
            .requestMatchers("/", "/index.html", "/style.css", "/app.js").permitAll()
            .requestMatchers("/h2-console/**").permitAll()
            .requestMatchers(HttpMethod.GET, "/api/students/**")
                .hasAnyRole("USER", "ADMIN")
            .requestMatchers("/api/students/**")
                .hasRole("ADMIN")
            .anyRequest().authenticated()
        )
        .headers(headers -> headers.frameOptions(frame -> frame.disable()))
        .httpBasic();

    return http.build();
}
```

---

## 🔍 Explanation

### 🔒 CSRF Disable
- REST API testing အတွက် disable လုပ်ထား

### 🌐 Public Access
- Frontend files → login မလို
- H2 console → open

### 👁️ Read Access
- USER / ADMIN → GET only

### ✏️ Write Access
- ADMIN → POST / PUT / DELETE

### 🔐 Default
- Other endpoints → login required

---

## 🧩 Role Diagram

```
USER  → GET only
ADMIN → FULL CRUD
```

## 🎯 Summary

- SecurityConfig = rules define
- FilterChain = request checker
- httpBasic = login
- hasRole = authorization
